Seditio 178

УСТАНОВКА

1. Скопировать файлы движка в нужный каталог сайта

2. Сделайте следующие директории и все их поддиректории доступными для записи

/datas/avatars
/datas/defaultav
/datas/photos
/datas/thumbs
/datas/resized
/datas/signatures
/datas/users

3. Зайти по адресу http://ваш_сайт/install

4. Следовать инструкциям установщика.

КОНФИГУРАЦИЯ И НАСТРОЙКА ДВИЖКА

Настройки находятся в файле /datas/config.php

По всем вопросам обращайтесь на сайт Seditio Team http://seditio.org
